# Analysis code for family history of cancer study

This repository contains the SAS and R code used in the study:

**Family history of cancer and risks of concordant and discordant cancers:
a nationwide cohort study of Korean women**

## Contents

### SAS
- Construction of study variables
- Cox proportional hazards regression models
- Concordant and discordant cancer analyses
- Incidence rates and incidence rate differences

### R
- `Figure1.R`: Generation of Figure 1

## Software

- SAS version 9.4
- R version 4.1.0

## Data availability

The individual-level data used in this study are not included in this
repository. Access to the National Health Insurance Service–National
Health Information Database requires formal authorization from the
National Health Insurance Sharing Service.

The SAS code requires access to the authorized analytic dataset.
Local directory paths and institution-specific library information
have been replaced with placeholders.
